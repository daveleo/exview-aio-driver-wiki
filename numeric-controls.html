Numeric Parameter Calculations

These are the control values developers usually need for automation: volume, brightness, contrast, saturation, hue, and RGB gains.

Formulas and Ranges

Control UI Range Protocol Data Byte Notes
Volume (C203) 0..100 proto = clamp(round(ui), 0, 100) -> 0x00..0x64 Special checksum rule: (0x5B + value) & 0xFF
Brightness (C21F) 0..100 proto = clamp(round(ui), 0, 100) -> 0x00..0x64 Special checksum rule: (0x77 + value) & 0xFF
Contrast (C217) 0..100 proto = clamp(round(ui), 0, 100) -> 0x00..0x64 Normal checksum sum rule
Saturation (C259) 0..100 proto = clamp(round(ui), 0, 100) -> 0x00..0x64 Normal checksum sum rule
Hue (C262) -50..+50 proto = clamp(round(ui + 50), 0, 100) -> 0x00..0x64
ui = proto - 50 		Normal checksum sum rule
Red gain (C223) 0..100 0x00..0x64 Normal checksum sum rule
Green gain (C227) 0..100 0x00..0x64 Normal checksum sum rule
Blue gain (C22B) 0..100 0x00..0x64 Normal checksum sum rule
